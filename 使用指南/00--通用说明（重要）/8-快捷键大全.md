
## 模块化OB快捷键
以下是win系统置，如果你是mac系统，则 Ctrl=cmd，Alt=Option；

| 快捷键          | 功能                 | 说明                                                                                                                                                           |
| ------------ | ------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Alt+1        | 嵌入Meta-bind模板      | 用于嵌入Assistants中的模块、模板                                                                                                                                        |
| Alt+2        | 插入2分栏              | 适用于所有情况，支持第三方插件                                                                                                                                              |
| Alt+3        | 插入3分栏              | 适用于所有情况，支持第三方插件                                                                                                                                              |
| Alt+4        | 插入callout注释        | 自动化了输入流程，与鼠标右键Inset>callouts（标注）效果一样                                                                                                                         |
| Alt+A        | 插入Admonition       | 比软件自带的callouts（标注）更强大                                                                                                                                        |
| Alt+C        | 重组笔记               | 将选中文字内容合并到其他笔记中                                                                                                                                              |
| Alt+D        | 展开/折叠canvas节点      | 在白板中展开或折叠选中节点的子节点                                                                                                                                            |
| Alt+E        | 打开emoji选中窗口        | 用于插入emoji                                                                                                                                                    |
| Alt+F        | 插入column分栏         | 适用于图片、文档分栏，或不支持第三方插件                                                                                                                                         |
| Alt+J        | 选择题型创建考题笔记         | 创建的笔记存放在`Documents/QuestionBank`                                                                                                                             |
| Alt+K        | 辅助工具选择器            | 选择批量替换、一键嵌入图片、中英互译等功能                                                                                                                                        |
| Alt+M        | 完形填空样式             | 选中文字，使用此命令，将选中文字替换为完形填空样式，例如<sub>这样</sub>                                                                                                                    |
| Alt+Shift+M  | 添加Media Browser代码块 | 用于嵌入库外文件夹以及文件                                                                                                                                                |
| Ctrl+shift+A | 一键切换完形填空答案         | 一键切换（显示、隐藏）所有完形填空答案                                                                                                                                          |
| Ctrl+shift+U | 开启/关闭完型填空功能        | 开启或关闭完型填空功能，以使用默认的下标样式                                                                                                                                       |
| Alt+P        | 插入批注               | 像word一样，在文档左侧、右侧（栏宽以外）插入批注                                                                                                                                   |
| Alt+Q        | 在当前文档插入模板          | 笔记抬头模块中“选择插入模板”按钮功能一致                                                                                                                                        |
| Alt+R        | 刷新笔记               | <font color="#ff0000">`[MB_UNLOADED] Embed`</font>出现时，使用此快捷键刷新                                                                                               |
| Alt+S        | 创建闪念笔记             | 捕获闪念，保存到新建文档或已有文档                                                                                                                                            |
| Alt+T        | 添加视频时间戳笔记          | 需要同时打开视频、笔记，支持B站和本地视频                                                                                                                                        |
| Alt+V        | AI对话               | 在任意位置呼出AI对话框                                                                                                                                                 |
| Alt+W        | 批量嵌入库外文件           | 将指定路径的文件夹、文件批量嵌入当前文档光标位置                                                                                                                                     |
| Alt+X        | 查看今日需要详述的笔记        | 查看今日（未来一周、逾期）需要详述的笔记                                                                                                                                         |
| Alt+Y        | 将带有标签的笔记移动至指定文件夹   | 此命令通过quickadd宏命令实现，移动文档排除了Assistants文件夹，但是需要*谨慎*移动                                                                                                           |
| Alt+←        | 切换左侧边栏             | 可[配合鼠标](https://www.xiaohongshu.com/explore/67fa302c0000000009038bd6?xsec_token=ABlNiuxqYxL3cpqvRwUfpdXcFbCiDA-ABmhSWor4xrv0c=&xsec_source=pc_user)实现快捷折叠或展开 |
| Alt+→        | 切换右侧边栏             |                                                                                                                                                              |
| Alt+shift+1  | 更新模块               | 请先更新使用指南，查看0-更新日志文档，再决定是否更新模块                                                                                                                                |
| Alt+shift+2  | 更新使用指南             | 建议定期更新                                                                                                                                                       |
| Alt+shift+B  | 在canvas白板中嵌入反向链接笔记 | 将具有共同主题的笔记，同时平铺在一张白板画布中，以进行思维导图、关联、编辑、整理等                                                                                                                    |
| Alt+shift+R  | 在项目中添加任务           | 快速添加dida任务                                                                                                                                                   |
| Ctrl+D       | 创建/打开今日日记          | 与左侧工具栏中的按钮功能一致                                                                                                                                               |
| Ctrl+K       | 快速创建笔记             | 输入文件夹，需要按住shift+enter                                                                                                                                        |
| Ctrl+P       | 打开命令面板             | 所有的命令都在这里                                                                                                                                                    |

以上快捷键涉及本系统的核心功能，因此建议**记住这些快捷键**：
- 使用以上快捷键，将提高本系统的使用效率和使用体验；
- 当然，为了提高使用体验，在[[homepage-主页|主页]]、[[homepage-侧页|侧页]]中，均设置了可视化功能按钮，按钮与快捷键具有同样的效果；
- 由于使用了不同的键盘，或许会导致快捷键不起作用，请排除是否外接键盘导致快捷键不起作用；
- 预设的快捷键或许与你电脑中自行安装的 app 快捷键冲突（目前尚未发现与系统快捷键冲突），若有冲突，请排除这些因素；
- 若快捷键仍然无法使用，请检查`设置>快捷键`是否已经存在这些快捷键，若不存在，大概率是同步或者拷贝时，`hotkeys.json`快捷键文档丢失或损坏造成，请自行检查。

## 已知的快捷键冲突/差异
- 打开开发者工具
	- mac系统:使用cmd+opt+i
	- win系统:使用ctrl+shift+i
- Alt(opt)+2：在mac、win系统中，与 pixpin 中的快捷键存在冲突，需要修改或关闭 pixpin 中的快捷键设置。
